## POB Share Code Spike 精度优化 — 75% → 92%

### 核心修复：Keystone 天赋节点 keystoneMod 缺失

**根因**：`_spike_step3_calc.py` 的 `load_tree_into_lua` 没有为 Keystone 类型节点创建 `keystoneMod` 字段。

**POB 桌面版机制**：
1. `PassiveTree.lua:474-476` 为 Keystone 节点创建 `node.keystoneMod = modLib.createMod("Keystone", "LIST", node.dn, "Tree:"..node.id)`
2. `CalcSetup.lua:1318` 调用 `buildModListForNodeList(env, allocNodes, true)` **不传** `includeKeystoneMods`，所以 Keystone 的实际 modList 被跳过
3. 但 `keystoneMod` (即 `mod("Keystone", "LIST", "Mind Over Matter")`) 无论 `includeKeystoneMods` 如何都会被添加
4. `CalcPerform.lua:66-76` 的 `mergeKeystones` 函数从 modDB 中找到 `"Keystone" LIST` mods，再从 `keystoneMap` 获取完整 modList 并添加到 modDB

**修复**：在 `_spike_step3_calc.py` 第 1167 行后添加：
```lua
node.keystoneMod = modLib.createMod("Keystone", "LIST", node.dn, "Tree:"..node.id)
```

**影响**：修复了 Mind Over Matter（MoM）基石效果缺失：
- `DamageTakenFromManaBeforeLife = 100`（All Damage from Mana before Life）
- `ManaRecoveryRate MORE -50%`（50% less Mana Recovery Rate）
